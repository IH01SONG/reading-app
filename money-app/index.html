앱 설치 가이드

앱처럼 편리하게 사용해 보세요!

안드로이드 (Chrome 브라우저)
  1. 앱처럼 사용하고 싶은 페이지에서 **Chrome 브라우저 메뉴** ( 또는 )를 탭하세요.
  2. 메뉴에서 **'앱 설치'** 또는 **'홈 화면에 추가'** 옵션을 찾아서 탭하세요.
  3. 확인 팝업이 나타나면 **'설치'** 버튼을 탭하면 홈 화면에 추가됩니다.
아이폰/아이패드 (Safari 브라우저)
  1. **Safari 브라우저**에서 앱처럼 사용하고 싶은 페이지를 엽니다. (다른 브라우저에서는 기능이 제한될 수 있습니다.)
  2. 하단의 **공유 버튼** ()을 탭하세요.
  3. 공유 메뉴에서 아래로 스크롤하여 **'홈 화면에 추가'** 옵션을 찾아서 탭하세요.
  4. 이름을 확인하고 **'추가'** 버튼을 탭하면 홈 화면에 아이콘이 생성됩니다.
